# Sign Maker by Chitose.City
这是一个基于HTML + JS的导视标牌生成器。

## 功能
* 通过组件系统直观的制作、编辑导视标牌
* 导出标牌为PNG / SVG格式图片
* 通过Add-on更改标牌风格
